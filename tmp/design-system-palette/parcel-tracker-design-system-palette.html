Parcel Tracker

Color palette grouped by hue family

This version is organized by actual hue family rather than by raw token category. The important part is that the system now has fewer real color families: neutrals, one accent family, one warning family, and one danger family. Supporting and less-used derived tokens are grouped separately below in a smaller section.
